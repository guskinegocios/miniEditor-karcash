# KarCard Generator 🚗💨

> **Usina de Artes para Venda Automotiva**

O **KarCard Generator** é uma aplicação web focada em produtividade extrema para revendedores de veículos. Ele transforma fotos comuns de pátio em anúncios de estúdio profissional para **Stories (Instagram/WhatsApp)** e **Feed (Poster)** em segundos.

## 🚀 Funcionalidades Premium

### ✨ Inteligência Artificial (AI)
- **Remoção de Fundo Nativa**: Isolamento automático do veículo utilizando IA diretamente no navegador. 
- **Privacidade e Custo Zero**: Todo o processamento é feito localmente (Client-side), sem envio de fotos para servidores externos e sem custos de API.

### 🖼️ Galeria de Fundos "Studio"
- **12 Versões Curadas**: Ambientes Industriais, Modernos, Luxury e Elite para valorizar cada perfil de carro.
- **Máscara Overlay Dinâmica**: Controle tonal e de opacidade do fundo para garantir que os textos neon sempre se destaquem (UX First).
- **Rotação de Fundo (Flip)**: Inversão horizontal dos ambientes para melhor composição estética.

### 🎨 Mini-Estúdio de Edição (Pós-Produção)
- **Ajustes de Imagem**: Sliders de **Brilho, Contraste, Saturação e Exposição** para harmonizar a iluminação do veículo com o novo fundo.
- **Hardware Accelerated**: Renderização via Canvas API garantindo fluidez total nos ajustes em tempo real.

### 📋 Controle Total de Layout
- **Dual Format**: Suporte completo para **Story (9:16)** e **Feed/Poster (4:5)** com configurações de texto independentes.
- **Transformações Livres**: Controle de **Zoom (Pinch/Slide)**, **Pan (Eixos X/Y)** e **Rotação** do veículo.
- **Tipografia KarCash**: Uso das fontes *Archivo* e *Montserrat* com controle de tamanho e peso para máxima legibilidade.

## 🛠️ Tecnologias Utilizadas

- **React + Vite**: Performance e desenvolvimento moderno.
- **TypeScript**: Robustez e escalabilidade do código.
- **Tailwind CSS**: Design system baseado na identidade KarCash.
- **@imgly/background-removal**: Processamento de IA local para extração de objetos.
- **HTML5 Canvas API**: Motor de renderização de alta definição para exportação fiel.

## 📦 Como Usar

1. **Subir Veículo**: Carregue a foto do carro e use a **IA** para remover o fundo.
2. **Escolher Cenário**: Na aba **Fundo**, selecione um ambiente premium.
3. **Calibrar Imagem**: Use os **Ajustes da Foto** para casar o brilho e a cor do carro com o cenário.
4. **Enquadrar**: Ajuste o **Zoom**, **Posição X/Y** e a **Rotação** do veículo.
5. **Dados e Preços**: Insira as informações do veículo. O sistema calcula a **Margem de Lucro** automaticamente.
6. **Exportar**: Clique em **"Baixar Arte"**. O sistema gerará um arquivo PNG pronto para postagem.
